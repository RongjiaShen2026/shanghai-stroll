1. 
```python
nx_unique_edges = len(set((u,v) for u,v in G.edges())) 
```
```python
(u, v) for u, v in G.edges()
```
这是一个生成式，意思是：
把 G.edges() 里面的每条边都整理成 (u, v) 这种二元组。

比如如果边是：
```
A -> B
A -> B
B -> C
```

生成出来就是：
```
(A, B), (A, B), (B, C)
```

### set(...) 会去重。所以：

```python
set((u, v) for u, v in G.edges())
```
会把重复边去掉。

```python

```
{nx_distance: .2f}：这是对变量 nx_distance 做格式化：

1. nx_distance 是你想显示的数值（通常是 float）。
2. : 后面是格式说明： .2f → 保留 两位小数 的浮点数。
